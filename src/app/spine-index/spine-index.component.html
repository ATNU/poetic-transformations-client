Spine Index Example

This page demonstrates a feature for future releases. A spine index has been automatically generated ahead of time using a cosine-similarity measure to compare all lines in all versions and identify lines that are similar enough to be considered a match. For this pilot version the cosine-similarity measure deemed acceptable for a match is 0.7. The program currently outputs a CSV file listing each spine index with the lines assigned to it.

In the example below, poem M1 line 20 has been selected.
Search the spine index CSV to find the location of that line in other versions.

Line ID: M1.20

Text: in the journey of an ant

Using the spine index with cosine similarity of 0.7, we are able to detect lines that are likely to be M1.20 or adaptations of that line
