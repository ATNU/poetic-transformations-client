Spine Index Example

This page demonstrates a feature for future releases. A spine index has been automatically generated using a cosine-similarity measure to compare all lines in all versions and identify lines that are similar enough to be considered a match. For this pilot version the cosine-similarity measure deemed acceptable for a match is 0.7. The program currently outputs a CSV file listing each spine index with the lines assigned to it.
